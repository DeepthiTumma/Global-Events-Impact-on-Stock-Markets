# Global Events Impact on Stock Markets

## Overview
An end-to-end Data Analytics project analyzing how global events affect stock market performance, volatility, and investor sentiment.

## Tools Used
- Python
- Pandas
- NumPy
- SQL
- Power BI
- Matplotlib
- Seaborn

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. SQL Analysis
4. Power BI Dashboard Development
5. Financial Insights & Risk Analysis

## Key Insights
- Impact of economic news on market returns
- Relationship between VIX and volatility
- Geopolitical risk analysis
- Sentiment score analysis
- Time series trend analysis

- EDA Notebook
